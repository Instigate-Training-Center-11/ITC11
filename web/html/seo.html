Title Tags - what the page is all about.

Meta Keywords Tags - meta name="keywords" content="key words" - define keywords for search engines.

Meta Description Tags - meta name="description" content="File discription" - specifies a description of the page.

Heading Tags (H1-H6) - HTML tags which are used to identify headings and subheadings within your content from other types of text.

Image Alt Attributes - img src="img.jpg" alt="text for an image" - used to embed an image in an HTML page and specifies an alternate text for an image.

Nofollow Attributess - rel=“nofollow” - tells search engines not to follow one specific outbound link.

Robots Meta Tag - meta name="robots" content="noindex" - tag or directive applies to search engine crawlers.

Link Tag - rel=”canonical” - way of telling search engines which version of a page you consider the main one and would like to be indexed by search engines and found by people.

Social Media Meta Tags - meta name="robots" content="noindex" - tag or directive applies to search engine crawlers.

Viewport Meta Tag - meta name="robots" content="noindex" - tag or directive applies to search engine crawlers.
